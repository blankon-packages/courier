Courier Socks 5 proxy client/server

This package implements a Sock5 proxy client/server, as defined in RFC 1928 and RFC 1929.

  • Access control: define IP address ranges allowed to proxy through the server.

  • Access control: import list of banned IP address range, in the P2P file format.

  • A "socksify" application wrapper script, that uses shared library tricks to try to redirect network connections from standalone application through the Socks 5 proxy server.

This proxy library was originally written for the Courier mail server, but it can be used independently, as a generic proxy client or server library. The Courier authentication library must be installed first, even if authentication is not needed, because the Socks 5 client/server uses some scripts and tools from that package. Additionally, the Net::CIDR Perl module is also required.
